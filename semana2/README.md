# Semana 2 · Backend 2

Material construido a partir de la guía **Implementación de autenticación y autorización en Backend con Spring Security**.

## Contenido de esta semana

### 1. teoria-jwt-interactiva
Sitio web estático para apoyar la explicación de:
- autenticación vs autorización
- stateful vs stateless
- estructura de JWT
- buenas prácticas de seguridad

### 2. empresa-x-jwt-security
Proyecto Spring Boot con:
- login que entrega JWT
- endpoints públicos y protegidos
- autorización por rol
- validación de Bearer token en cada solicitud

## Uso sugerido en clase

1. Abrir la web interactiva para introducir la teoría.
2. Ejecutar el proyecto Spring Boot.
3. Mostrar login, token y acceso a endpoints protegidos.
4. Comparar acceso de `usuario_demo` y `admin_demo`.
