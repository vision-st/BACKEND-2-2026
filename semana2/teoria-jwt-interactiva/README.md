# teoria-jwt-interactiva

Sitio estático para explicar la teoría principal de la semana 2 de Backend 2.

## Contenido

- autenticación vs autorización
- stateful vs stateless
- estructura de un JWT
- buenas prácticas de seguridad
- mini chequeo interactivo

## Uso

Abrir `index.html` en el navegador.

## Objetivo pedagógico

Servir como apoyo visual para la clase antes o durante la demostración del proyecto Spring Boot con JWT.
