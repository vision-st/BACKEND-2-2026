# BACKEND-2-2026

Repositorio de proyectos semanales de la asignatura **Backend 2**, con foco en desarrollo con **Spring Boot** y **Spring Security** durante el año 2026.

## Objetivo del repositorio

Este repositorio organiza el trabajo práctico desarrollado semana a semana, manteniendo una estructura clara y consistente. Cada carpeta `semanaX` representa una semana de clases y dentro de ella se almacenan los proyectos realizados.

## Estructura general

```text
BACKEND2/
├── semana1/
├── semana2/
├── semana3/
├── semana4/
├── semana5/
├── semana6/
├── semana7/
├── semana8/
├── semana9/
└── semana10/
```

## Criterio de organización

- Cada semana puede contener uno o más proyectos según la actividad desarrollada.
- Cada proyecto incluye su propio `README.md` con explicación técnica y contenido teórico.
- El repositorio excluye archivos generados por IDEs, carpetas de compilación y configuraciones locales innecesarias.

## Estado actual

- `semana1`: contiene el proyecto **empresa-x-security**.
- `semana2`: contiene una web interactiva de teoría JWT y el proyecto **empresa-x-jwt-security**.
- Las semanas restantes quedan preparadas para futuros ejercicios y evaluaciones.

## Tecnologías base

- Java 17+
- Spring Boot
- Spring Security
- Maven
- IntelliJ IDEA
